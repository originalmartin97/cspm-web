# Development Log - February 2, 2026

## Major Project Restructure & Refactoring

### Overview

Performed a comprehensive refactoring of the entire project structure to improve code organization, maintainability, and security. This was a breaking change that reorganized most of the codebase.

---

## Security Improvements

### Sensitive Data Protection

1. **Personal Data Isolation**
   - Moved colleague personal information (names, photos, bios) to `_private/` folders
   - These folders are now gitignored and will not be uploaded to the repository
   - Created template files for safe reference

2. **Git History Cleanup**
   - Used `git-filter-repo` to completely remove sensitive files from git history
   - Removed colleague images and personal data from all historical commits
   - Force push required to update remote

3. **New Naming Convention for Sensitive Files**
   - Files with `_private.` prefix are automatically gitignored
   - Pattern: `_private.<description>.<ext>` (e.g., `_private.colleagues.js`)
   - Folders named `_private/` are also gitignored
